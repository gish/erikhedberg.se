Vad jag kan

Min expertis ligger inom frontendutveckling.

Webbsidor och webbappar behöver en bra användarupplevelse. Jag flyttar designen från ritbordet till webbläsaren och ger lösningen den struktur som behövs för att underhålla den.

Min favoriteditor är Sublime Text. Alla JavaScript laddas via RequireJS, skapar objekt i Backbone.js, manipulerar DOM med jQuery och får hjälp av Underscore.js hjälpfunktioner. CSS kompileras från SASS, där det har kryddats i Compass.

Vem jag är
